# iconfont for flutter

## 说明

⚠️ 目前仅针对[阿里巴巴矢量图标库](https://www.iconfont.cn/)进行适配！

## 使用

右键点击 `json` 文件后，会出现 `generate dart` 选项，使用该选项会在项目根目录的 `lib/widgets` 中生成 `dart` 文件

> 需要在 `pubspec.yaml` 中配置 `fonts` 字段
